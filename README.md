# ✍️ Генератор продающих постов для курсов

Flask-приложение, которое по ссылке на страницу курса генерирует продающий пост для соцсетей через OpenRouter API (бесплатные LLM-модели). Пользователь выбирает настроение поста, а нейросеть пишет текст с учётом настроек стиля из `voice.md`.

## Возможности

- Загрузка страницы курса по URL и автоматическое извлечение описания
- 6 настроений поста: деловой, вдохновляющий, непринуждённый, срочный, академический, юмористический
- Генерация через OpenRouter (free-модели, fallback при недоступности)
- Настройка стиля без перезапуска: тон, длина, эмодзи, заголовок, формат, CTA (файл `voice.md`)
- Плавный интерфейс с индикацией загрузки и уведомлением о копировании
- Markdown-разметка в ответе преобразуется в HTML (жирный, курсив)

## Быстрый старт

```bash
# Установить зависимости
pip install -r requirements.txt

# Указать API-ключ OpenRouter (получить: https://openrouter.ai/keys)
set OPENROUTER_API_KEY=sk-or-v1-...
# Либо создать файл .env и записать в него:
# OPENROUTER_API_KEY=sk-or-v1-...

# Запустить
python app.py
```

Открой `http://127.0.0.1:5000`.

## Настройка стиля (voice.md)

Редактируй `voice.md` — изменения применяются сразу, перезапуск не нужен.

| Параметр | Описание | Пример |
|----------|----------|--------|
| `TON` | Общий тон | профессиональный, дружелюбный, дерзкий |
| `LENGTH` | Длина поста | короткий (до 150 слов), средний (300–500) |
| `EMOJI` | Сколько эмодзи | без эмодзи, умеренное (2–4), активное (5–8) |
| `HEADLINE` | Стиль заголовка | вопрос, громкое обещание, интрига |
| `FORMAT` | Формат подачи | сплошной текст, список с буллитами, история |
| `CTA` | Призыв к действию | мягкий, прямой, срочный, без CTA |

## Выбор модели

В `app.py` (строки 25–26):

```python
DEFAULT_MODEL = "meta-llama/llama-3.3-70b-instruct:free"
FALLBACK_MODEL = "openrouter/free"
```

Список бесплатных моделей: https://openrouter.ai/models?output_modalities=text

## API

`POST /generate` — сгенерировать пост.

**Тело запроса (JSON):**
```json
{
  "url": "https://example.com/course",
  "mood": "inspirational"
}
```

**Успешный ответ:**
```json
{
  "success": true,
  "post": "✨ **Заголовок**\n\nТекст поста..."
}
```

## Используемые технологии

- **Flask 3.x** — веб-фреймворк
- **Requests + BeautifulSoup4** — загрузка и парсинг страниц
- **OpenRouter API** — единый доступ к LLM (Chat Completions)
- Чистый HTML/CSS/JS (без фронтенд-фреймворков)
