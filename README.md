# ✍️ Генератор продающих постов для курсов

Flask-приложение: по ссылке на курс генерирует продающий пост через OpenRouter, сохраняет в избранное, публикует во ВКонтакте сейчас или по расписанию.

## Возможности

- 🔗 Загрузка страницы курса по URL, извлечение описания
- 🎭 6 настроений поста: деловой, вдохновляющий, непринуждённый, срочный, академический, юмористический
- 🤖 Генерация через OpenRouter (free-модели)
- 📤 Публикация во ВКонтакте (стена сообщества)
- 📅 Отложенные посты с выбором времени
- ⭐ Сохранение постов в избранное
- 🎨 Настройка стиля через `voice.md` (без перезапуска)

## Быстрый старт

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Создать файл .env в папке проекта (рядом с app.py)
#    и записать в него ключи:
#    OPENROUTER_API_KEY=sk-or-v1-твой_ключ
#    VK_ACCESS_TOKEN=твой_токен
#    VK_GROUP_ID=123456789

# 3. Запустить
python app.py
```

Открой **http://127.0.0.1:5000**.

## Как подключить ВКонтакте

### 1. Создай приложение VK

Перейди на https://vk.com/apps?act=manage → **Создать приложение** → тип **Standalone**.

### 2. Получи токен доступа

Открой в браузере:

```
https://oauth.vk.com/authorize?client_id=ID_ПРИЛОЖЕНИЯ&scope=wall,groups,offline&response_type=token&redirect_uri=https://oauth.vk.com/blank.html
```

Где `ID_ПРИЛОЖЕНИЯ` — ID из шага 1.
После авторизации в адресной строке будет `access_token=...`.

### 3. Найди Group ID

Открой сообщество → **Управление** → **Работа с API** → **ID сообщества**.

### 4. Запиши в `.env`

Файл `.env` лежит в корне проекта и автоматически загружается при запуске (`python-dotenv`).

```
VK_ACCESS_TOKEN=полученный_токен
VK_GROUP_ID=айди_сообщества (без минуса)
OPENROUTER_API_KEY=sk-or-v1-твой_ключ
```

Токен должен иметь права `wall` и `groups`. Параметр `offline` даёт бессрочный доступ.

## API

| Метод | Путь | Описание |
|-------|------|----------|
| `POST` | `/generate` | Сгенерировать пост |
| `POST` | `/api/publish` | Опубликовать в VK |
| `GET` | `/api/vk-status` | Статус VK-подключения |
| `GET` | `/api/favorites` | Список избранного |
| `POST` | `/api/favorites` | Добавить в избранное |
| `DELETE` | `/api/favorites/<id>` | Удалить из избранного |
| `GET` | `/api/scheduled` | Список отложенных |
| `POST` | `/api/schedule` | Запланировать пост |
| `DELETE` | `/api/scheduled/<id>` | Удалить из расписания |

## Настройка стиля

Редактируй `voice.md` — изменения без перезапуска:

| Параметр | Описание | Пример |
|----------|----------|--------|
| `TON` | Общий тон | профессиональный, дерзкий |
| `LENGTH` | Длина | средний (300–500 слов) |
| `EMOJI` | Эмодзи | умеренное (2–4) |
| `HEADLINE` | Заголовок | громкое обещание, интрига |
| `FORMAT` | Формат | список с буллитами, история |
| `CTA` | Призыв к действию | прямой, срочный |

## Выбор модели

В `app.py` (найди строки `DEFAULT_MODEL` / `FALLBACK_MODEL`):

```python
DEFAULT_MODEL = "meta-llama/llama-3.3-70b-instruct:free"
FALLBACK_MODEL = "openrouter/free"
```

Актуальный список free-моделей: https://openrouter.ai/models?output_modalities=text
